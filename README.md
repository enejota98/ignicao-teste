# Teste Ignição Digital

Front-End da Página estática do teste proposto pela Ignição Digital

# Tecnologias
-NodeJS
-ReactJS
-Yarn
-Material UI

# Baseado em
https://www.figma.com/file/9LFpKshMWD0P056KBkSME9/Teste-Igni%C3%A7%C3%A3o-Digital?node-id=0%3A1


# Como executar
Instale o NodeJS e dentro do repositório execute o comando "npm install" ou "yarn create" caso opte por essa tecnologia. Em seguida rode o sistema local com "npm start" ou "yarn start".
